Examples

  • Packer Examples

    Manual Vagrant Box Builders: https://developer.hashicorp.com/packer/plugins/builders/vagrant#example
    cd /vagrant/packer.d
    packer build centos-packer.json
    cd /vagrant/packer.d
    packer build ubuntu-kde.json

    Terraform Examples

    cd /vagrant/terraform.d
    terraform init
    terraform apply
    And then open in your host browser http://docker.lh

    Vault UI

    Start Server: sudo vault server -config=/etc/vault/config.hcl
    Remove Seal Data If Needed: sudo rm -rf /var/lib/vault
    Manual: https://developer.hashicorp.com/vault/tutorials/getting-started/getting-started-ui
    vault kv get secret/my_projects
    vault kv get -field=foo secret/my_projects

    Ansible Examples

    ansible-galaxy install --roles-path /vagrant/ansible.d/playbook/roles -r /vagrant/ansible.d/requirements.yml --force
    sudo ansible-playbook -i /vagrant/ansible.d/inventory /vagrant/ansible.d/playbook/install-jenkins-slaves.yml

    Install Nagios NRPE Agents with Ansible

    sudo ansible-playbook -i /vagrant/ansible.d/inventory /vagrant/ansible.d/playbook/install-nagios-nrpe-agents.yml
    sudo /usr/lib64/nagios/plugins/check_nrpe -H 10.3.3.2

    Setup Default JRE Version

    sudo alternatives --config java

    Start/Restart Jenkins Swarm Client on Slave Host

    sudo service jenkins-swarm-client restart

    Start/Restart Nagios Server (Check Configs Before)

    sudo nagios -v /etc/nagios/nagios.cfg
    sudo service nagios restart

    How to Fix Guest Additions to Can Mount Shared Folders

    sudo yum install elfutils-libelf-devel
    cd /opt/VBoxGuestAdditions-*/init
    sudo ./vboxadd setup
