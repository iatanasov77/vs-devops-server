• Install Jenkins Slaves Agents with Ansible

    ansible-playbook -i /vagrant/ansible.d/inventory /vagrant/ansible.d/playbook/install-jenkins-slaves.yml -e "jenkins_master='{{ devopsConfig?.hostAddress }}' jenkins_swarm_client_version='{{ devopsConfig?.subsystems.jenkins.swarmVersion }}'"

    Setup Default JRE Version

    sudo alternatives --config java

    Start/Restart Jenkins Swarm Client on Slave Host

    sudo service jenkins-swarm-client restart
