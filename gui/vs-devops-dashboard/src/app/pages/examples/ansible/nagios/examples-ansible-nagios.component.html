• Install Nagios NRPE Agents with Ansible

    sudo ansible-playbook -i /vagrant/ansible.d/inventory /vagrant/ansible.d/playbook/install-nagios-nrpe-agents.yml
    sudo /usr/lib64/nagios/plugins/check_nrpe -H 10.3.3.2

    Start/Restart Nagios Server (Check Configs Before)

    sudo nagios -v /etc/nagios/nagios.cfg
    sudo service nagios restart
