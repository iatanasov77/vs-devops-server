• Install Ansible Roles Requirements

    ansible-galaxy install --roles-path /vagrant/ansible.d/playbook/roles -r /vagrant/ansible.d/requirements.yml --force

  • Install Jenkins Slaves Agents with Ansible

    sudo ansible-playbook -i /vagrant/ansible.d/inventory /vagrant/ansible.d/playbook/install-jenkins-slaves.yml -e "jenkins_master='10.3.3.3' jenkins_swarm_client_version='3.40'"

    Setup Default JRE Version

    sudo alternatives --config java

    Start/Restart Jenkins Swarm Client on Slave Host

    sudo service jenkins-swarm-client restart

  • Install Nagios NRPE Agents with Ansible

    sudo ansible-playbook -i /vagrant/ansible.d/inventory /vagrant/ansible.d/playbook/install-nagios-nrpe-agents.yml
    sudo /usr/lib64/nagios/plugins/check_nrpe -H 10.3.3.2

    Start/Restart Nagios Server (Check Configs Before)

    sudo nagios -v /etc/nagios/nagios.cfg
    sudo service nagios restart
